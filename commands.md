### Start Python Env:
```
source d:/GitHub/file-reader/.venv/Scripts/activate
```

### Stop Python Env:
```
deactivate
```

### Build Docker Container:
```
docker build -t your-image-name .
```

### Run Docker Container:
```
docker run your-image-name
```
### Install pytest:
```
pip install pytest
```

### Run Python Test:
```
pytest -v --tb=short readertest.py
```
